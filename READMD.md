# 🏠 智能家居控制 APP

## 📋 项目概述

智能家居控制 APP 是一个基于 Android 平台的完整物联网解决方案，提供设备控制、情景模式、语音控制等核心功能，支持多用户协作和远程访问。

### 🎯 核心目标

- **智能化控制**：实现家居设备的智能化和自动化控制
- **多模态交互**：支持触控、语音、自动化多种交互方式
- **云端协同**：本地控制与云端服务的无缝结合
- **安全可靠**：企业级的安全保障和权限管理

## ✨ 功能特性

### 🎮 设备管理

- **智能发现**：自动扫描局域网设备，支持多种发现协议
- **一键配网**：SmartConfig 智能配网，快速设备接入
- **实时控制**：设备状态实时同步，毫秒级响应控制
- **分组管理**：按房间、功能进行设备分组管理

### 🤖 情景模式

- **多种触发**：时间、设备状态、语音、手动触发
- **复杂逻辑**：支持条件判断、延迟执行、批量操作
- **智能联动**：多设备协同工作，实现场景自动化
- **模板库**：丰富的预置场景模板

### 🎤 语音控制

- **语音识别**：集成 Android 原生和云端语音识别
- **自然交互**：支持自然语言指令解析
- **离线支持**：常用指令离线识别，快速响应
- **语音反馈**：TTS 语音播报操作结果

### 👥 多用户系统

- **家庭共享**：家庭成员协同控制，权限分级管理
- **角色权限**：管理员、成员、访客多级权限控制
- **操作审计**：完整的操作日志和安全审计
- **数据隔离**：多家庭数据安全隔离

### 🌐 远程访问

- **云端中转**：通过云服务器实现远程控制
- **实时同步**：多客户端状态实时同步
- **离线队列**：网络异常时指令缓存，恢复后执行
- **推送通知**：设备状态变化实时推送

## 🏗️ 技术架构

### 系统架构图

```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   Android APP   │◄──►│   云服务平台    │◄──►│   硬件设备层    │
│   (Java)        │    │ (Spring Boot)   │    │   (ESP32)       │
└─────────────────┘    └─────────────────┘    └─────────────────┘
         │                       │                       │
    ┌─────┴─────┐           ┌─────┴─────┐           ┌─────┴─────┐
    │ MVVM架构  │           │ 微服务    │           │ 传感器    │
    │ 模块化设计 │           │ 消息队列  │           │ 执行器    │
    └───────────┘           └───────────┘           └───────────┘
```

### 技术栈

#### 移动端 (Android)

- **开发语言**：Java
- **架构模式**：MVVM + 模块化
- **核心框架**：Jetpack (ViewModel, LiveData, Room)
- **网络通信**：Retrofit + OkHttp + MQTT Paho
- **数据库**：Room + SQLite
- **语音识别**：Android Speech API

#### 服务端

- **后端框架**：Spring Boot + Spring Security
- **数据库**：MySQL + Redis
- **消息队列**：EMQX (MQTT Broker) + RabbitMQ
- **实时通信**：WebSocket + STOMP
- **部署**：Docker + Nginx

#### 硬件端

- **主控**：ESP32 开发板
- **通信**：WiFi + MQTT 协议
- **传感器**：温湿度、光照、人体感应等
- **执行器**：继电器、舵机、LED 等

## 🚀 快速开始

### 环境要求

- Android Studio Arctic Fox+
- JDK 11+
- Android SDK 30+
- Gradle 7.0+

### 项目结构

```
smart-home-app/
├── app/                 # 主应用模块
├── lib-core/           # 核心业务逻辑
├── lib-network/        # 网络通信
├── lib-mqtt/          # MQTT消息处理
├── lib-database/      # 数据持久化
└── lib-common/        # 公共工具
```

### 构建步骤

1. **克隆项目**

```bash
git clone https://github.com/your-username/smart-home-app.git
cd smart-home-app
```

2. **配置环境**

- 在 `local.properties` 中配置 Android SDK 路径
- 在 `app/src/main/res/values/config.xml` 中配置服务器地址

3. **构建运行**

```bash
./gradlew assembleDebug
# 或通过Android Studio直接运行
```
